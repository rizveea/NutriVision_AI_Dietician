# Data Directory

Required file: **full_recipes_with_nutrition_and_cnn_class-1.csv**

## Dataset Contents

- 2M+ recipes with nutrition data
- Mapped to 11 CNN food categories
- Includes: Calories, Protein, Fat, Carbs, Sodium, Cholesterol, Fiber

## Download

1. Access from team Google Drive (group project data)
2. Or create from raw datasets:
   - [Food-11 Images](https://www.kaggle.com/datasets/trolukovich/food11-image-dataset)
   - [USDA Nutrition](https://www.kaggle.com/datasets/demomaster/usda-national-nutrient-database)
   - [Recipe DB](https://www.kaggle.com/datasets/thedevastator/better-recipes-for-a-better-life)

## File Structure

```csv
recipe_name,cnn_category,ingredients,healthiness_score,Calories,Protein,TotalFat,Carbohydrate,Cholesterol,Sodium,Fiber
```
